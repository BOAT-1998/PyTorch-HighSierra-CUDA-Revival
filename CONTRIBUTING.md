# Contributing Guide

Thank you for preserving this historic environment.

## Development Standards

- Python style: black, flake8 (see `pyproject.toml`).
- Tests: `pytest` in `tests/` with CPU fallback; GPU tests are optional/mocked.
- Docs: update `README.md` and `docs/*` with clear reproducible steps.

## Pull Requests

- Include a description, rationale, and references.
- Link to CI results and environment probes.
- Update `CHANGELOG.md` and `RELEASE_NOTES.md`.
